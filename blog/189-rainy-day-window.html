비 오는 날 창밖을 볼 때의 기분

비 오는 날 창밖을 볼 때 느껴지는 묘한 감정과 생각들. 빗소리와 함께하는 혼자만의 시간에 대한 일상적인 철학

창밖의 비

오늘은 하루 종일 비가 오더라고. 아침에 일어났을 때부터 지금까지 계속 내리고 있어. 원래는 나갈 계획이 있었는데, 이런 날은 그냥 집에 있는 게 나은 것 같아서 창가에 앉았어. 창문 너머로 비가 내리는 걸 보고 있으니까 시간 가는 줄 모르겠더라고.

비가 내리는 모습을 볼 때마다 드는 생각이 있는데, 이게 왜 이렇게 멍하니 보게 되는 건지 모르겠어. 특별한 게 있는 것도 아닌데, 그냥 계속 보게 되는 거야. 빗방울이 유리에 닿아서 흘러내리는 모습, 바람에 비가 삐뚤게 내리는 모습, 물웅덩이에 떨어지는 파문 같은 거.

생각핸지는 모르겠는데, 비 오는 날의 창밖은 평소랑 다른 세상처럼 느껴져. 색감도 다르고, 분위기도 다르고. 마치 사진 필터를 씌운 것 같은 느낌이랄까. 모든 게 좀 흐릿하고, 은은하고, 조용해 보여. 사람들도 다 어디론가 숨은 것 같고.

빗소리의 중독성

비 오는 날 가장 먼저 느껴지는 건 빗소리인 것 같아. 규칙적이지 않은 소리가 계속 들리는 거야. 때로는 부드럽게, 때로는 거세게. 근데 이게 생각보다 집중력을 방해하지 않더라고. 오히려 더 집중이 되는 것 같기도 해.

나는 가끔 빗소리를 ASMR처럼 듣기도 해. 인터넷에서 찾아서 틀어놓는 거지. 근데 진짜 비가 내릴 때 듣는 소리랑은 좀 다른 것 같아. 인공적인 소리는 너무 균일하거든. 진짜 비는 예측할 수 없는 패턴이 있어서 그게 더 편한 것 같아.

빗소리를 들으면서 생각핸지는 모르겠는데, 이게 뭔가 위로가 되는 것 같기도 해. 세상이 나한테 "괜찮아, 천천히 핚 돼"라고 말하는 것 같은 느낌이랄까. 바쁘게 살아야 한다는 압박감이 조금은 누그러지는 것 같아.

근데 또 재밌는 건, 빗소리가 너무 커지면 오히려 불편해진다는 거야. 적당히 들려야 좋은 거지, 폭우가 쏟아지면 조금 무섭기도 하거든. 적당함이라는 게 참 중요한 것 같아. 모든 것에 다 적용되는 거 같아.

유리창 너머의 세계

창문이 있어서 다행인 것 같아. 비 오는 날에 집에 있으면서 밖을 볼 수 있다는 게. 유리창이 있으니까 비를 직접 맞지 않으면서도 비의 풍경을 즐길 수 있거든. 이게 약간은 겁쟁이 같은 기분이 들기도 해.

유리창에 맺힌 물방울들을 볼 때마다 느껴지는 게 있어. 각각의 물방울이 다 다른 길을 그리면서 흘러내리잖아. 어떤 건 빨리 가고, 어떤 건 천천히 가고, 어떤 건 중간에 합쳐지기도 하고. 이게 사람이랑 좀 비슷한 것 같아.

생각핸지는 모르겠는데, 비 오는 날 창밖을 볼 때는 시간이 이상하게 느껴져. 한 시간을 봤는데도, 생각보다 시간이 안 간 것 같은 느낌이 들 때가 있어. 반대로 멍하니 보다 본전 시간이 훅 지나가기도 하고. 시간이란 게 원래 그런 거 같거든.

가끔은 창문을 조금 열어보기도 해. 비가 안 들어오는 각도라면 괜찮거든. 그러면 공기 냄새가 확 다른다는 걸 알 수 있어. 비 냄새라고 해야 하나, 흙 냄새가 나면서 상쾌한 냄새가 나는 것 같아. 이걸 petrichor라고 하나? 어쨌든 좋은 냄새야.

멈춰있는 것 같은 기분

비 오는 날은 왠지 세상이 멈춘 것 같은 기분이 들어. 사람들이 움직임이 줄어들고, 차들도 천천히 가고, 모든 게 좀 느려지거든. 이런 날은 나도 자연스럽게 느려지는 것 같아.

근데 이게 나쁜 건 아닌 것 같아. 평소에는 너무 빨리 돌아가는 세상에서 잠깐 쉬는 느낌이랄까. 비가 내리는 동안만큼은 뭔가를 해야 한다는 강박에서 벗어날 수 있는 것 같아. 그냥 있어도 되는 시간.

생각핸지는 모르겠는데, 비 오는 날의 이 느린 느낌이 왠지 그리울 때가 있어. 화창한 날에는 오히려 불안하게 느껴지거든. 뭔가 더 해야 할 것 같고, 나가야 할 것 같고. 근데 비 오는 날은 그런 게 없어. 그냥 있는 것만으로도 충분한 것 같아.

물론 비 오는 날에도 나가서 일해야 하는 사람들은 많아. 나만 이렇게 여유롭게 지내는 거지. 그래서 조금은 미안한 마음도 들기도 해. 내가 이렇게 편하게 있어도 되나 싶은. 근데 또 생각핸지는 모르겠는데, 그건 내가 통제할 수 있는 게 아니니까 너무 신경 쓰지 않기로 했어.

추억과 비

비 오는 날에는 왠지 추억이 많이 떠올라. 특정한 사건이 아니라, 그냥 분위기 같은 거야. 어렸을 때 비 오는 날의 기분이라든지, 예전에 비 오는 날 만났던 사람들의 얼굴이라든지. 왜 비와 연결되어 있는지 모르겠는데, 자꾸 생각나는 거야.

이게 뇌가 그렇게 작동하는 건가 봐. 특정한 감각(소리, 냄새, 시각)이 있으면 관련된 기억을 같이 끌어올리는 거지. 비는 특히 그런 것 같아. 빗소리만 들어도 어떤 순간이 떠오를 때가 있거든.

근데 이런 추억들이 다 좋은 건 아니야. 슬픈 기억도 떠오르고, 후회되는 기억도 떠올라. 그래도 그냥 다 받아들이는 편이야. 그게 나의 일부니까. 비 오는 날에는 이런 감정들도 자연스러운 것 같아.

가끔은 아무 생각도 안 하고 그냥 멍하니 있고 싶을 때가 있어. 근데 그게 잘 안 되거든. 뇌가 계속 뭔가를 생산하는 것 같아. 비 오는 날에도 마찬가지야. 그냥 창밖을 봐도 생각이 끊이지 않아. 이건 어쩔 수 없는 거 같아.

비가 그치면

비가 그치면 세상이 다시 원래대로 돌아가는 것 같아. 사람들이 바쁘게 움직이고, 차들도 빨리 달리고. 마치 비가 내리는 동안 잠시 멈췄던 세상이 다시 작동하는 거지. 나도 그 흐름에 다시 들어가야 하고.

근데 또 생각핸지는 모르겠는데, 비가 그치고 나면 뭔가 새로워진 느낌이 들어. 공기가 깨끗해진 것 같고, 모든 게 씻겨난 것 같고. 이게 상징적인 의미가 있는 건 아닌데, 그냥 그런 느낌이 들어.

비가 그친 후에 창밖을 별병 풍경도 달라져 있어. 물방울이 맺혀 있고, 땅이 젖어 있고, 하늘이 맑아져 있고. 마치 비가 모든 걸 정리해준 것 같은 느낌이랄까. 이런 변화가 반복되는 게 자연의 순리인 거 같아.

나도 비가 그치고 나면 다시 일상으로 돌아가야지. 할 일을 하고, 사람들을 만나고, 바쁘게 살아야지. 근데 지금 이 순간만큼은 그냥 이렇게 있고 싶어. 비 오는 창밖을 볼면서 뭔가를 생각하거나, 아무것도 생각하지 않거나.

마무리

비 오는 날 창밖을 볼 때의 기분은 참으로 묘해. 슬프지도 않고, 기쁘지도 않고, 그냥 뭔가 중간에 있는 감정이랄까. 근데 그게 나쁘지는 않은 것 같아. 인생에도 이런 날이 필요하지 않을까.

글을 쓰면서 보니까 비가 조금 약해진 것 같아. 곧 그치려나 봐. 비가 그치면 다시 평소의 내가 되어야겠지. 근데 지금 이 순간의 감정은 어딘가에 저장핸지는 모르겠어. 다음에 비가 올 때 다시 꺼낼 수 있도록.

여기까지 쓰고 보니까 생각보다 많은 걸 적었네. 원래 이런 감정은 평소에도 느끼는데, 잘 정리가 안 되거든. 글로 쓰니까 조금은 선명해지는 느낌이야. 비 오는 날 창밖을 보는 의미가 뭔지는 모르겠지만, 일단 이런 시간이 필요한 것 같아.

비가 그치기 전에 한번 더 창밖을 보고 올게. 오늘의 비는 어떤 모습일까. 모르겠어. 그냥 보는 거지. 아마도 비슷한 모습일 텐데, 또 뭔가 달라져 있을 수도 있고. 그게 비 오는 날의 매력인 것 같아. 매번 같은 것 같지만, 사실은 전부 다르거든.
