비 오는 날 창밖을 보다가 문득 생각한 것들

아침에 눈을 떴는데 하늘이 회색빛이더라고. 창문에 가까이 가서 보니까 물방울들이 유리 위로 미끄러지고 있었어. 이런 날은 특별히 할 일이 없어도 괜찮은 것 같아. 마치 날씨가 대신 "오늘은 천천히 복낼래?" 라고 말해주는 것 같은 느낌이랄까.

손에 잡히는 컵이 따뜻했어. 뭘 마시고 있는지는 잘 기억이 안 나. 그냥 뜨거운 게 손에 닿아있고, 창밖에는 비가 내리고 있으니까 뭔가 마음이 차분해지는 거야. 사람이 이렇게 아무것도 안 하고 있으면 이상한 생각이 드는 것 같아. 지난주에 했던 말들도 생각나고, 못 했다고 아쉬워했던 일들도 떠오르고.

근데 오늘은 그런 것들이 멀리 있는 것처럼 느껴졌어. 비 소리가 생각을 좀 멀리 복내주는 것 같아. 창밖을 지나가는 사람들 우산을 쓰고 급하게 걸어가는 것도 보였어. 다들 어디론가 가야 하는구나 싶으면서도, 나는 여기 서서 비를 구경하고 있구나 싶었어. 딱히 누구를 부러워하거나 그런 건 아니었어. 그냥 각자의 속도가 있는 것 같은 느낌?

어제 밤에는 좀 뒤척였었거든. 왜 그랬는지는 모르겠는데, 잠이 안 오더라고. 그래서 이것저것 생각하다가 새벽에 잠든 것 같아. 그런데 오늘 아침에 비를 보니까 어젯밤의 불안한 마음이 좀 가라앉은 것 같아. 꼭 해결해야 하는 문제가 사라진 것도 아니고, 달라진 것도 없는데 말이야.

비가 내리는 소리가 집중력을 떨어뜨리는 것 같으면서도 반대로 어떤 생각은 더 선명하게 해주는 것 같아. 요즘 내 머릿속이 잠깐잠깐 비어있는 느낌이 들 때가 있거든. 뭔가 채워야 할 것 같은데 뭘 채워야 할지 모르겠는 그런 기분. 근데 오늘은 그냥 그대로 둬도 괜찮을 것 같았어. 비가 오니까.

창문에 맺힌 물방울들이 서로 합쳐지면서 아래로 흐르는 게 재밌더라고. 작은 것들이 모여서 더 커지고, 그러다가 어느 순간 낙하하는 것 같아. 사람도 좀 그런 것 같기도 하고. 작은 경험들이 모여서 지금의 내가 되었으니까. 근데 그게 좋은 것만은 아니잖아. 때로는 안 좋은 기억들도 섞여 있고.

그래도 오늘은 그런 생각들이 그냥 스쳐 지나갔어. 뭐랄까, 비가 내리니까 마음에 생긴 창문도 조금 열리는 것 같은 느낌? 평소에는 답답하게 닫혀 있던 것들이 습기를 머금고 부드러워지는 것 같아. 이상한 비유인가. 나도 내가 뭔 말을 하는지 잘 모르겠네.

어쨌든 오늘은 뭔가 큰 결심을 하거나 그런 날은 아닌 것 같아. 그냥 창밖을 볼 수 있는 시간이 있어서 좋았어. 비가 그치면 다시 바쁜 일상으로 돌아가겠지만, 지금 이 순간만큼은 그냥 이렇게 있어도 되는 것 같아. 아무것도 증명하지 않아도 되고, 누군가에게 보여줄 필요도 없고.

나중에 비가 그치면 밖에 나가서 바람 쐬는 것도 좋을 것 같아. 지금은 그냥 이렇게.
